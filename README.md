# Brickyard RaMin0

The vehicle production factory.

## Running

To start the Docker container(s), run the following command.

  `docker-compose up -d`

Then navigate to http://localhost:8080.

## Future Improvements

* Allow the `PORT` to be configured at runtime (currently hardcoded to `8080`).
